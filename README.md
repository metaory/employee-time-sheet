# Timesheet

Printable monthly timesheet with Gregorian and Persian calendars

Fill start, end, and overtime per day. Pick month, year, locale, and first weekday. Print from the browser.

## Dev

```bash
pnpm install
pnpm dev
```

```bash
pnpm build
pnpm preview
```

## Deploy

Push to `main`. GitHub Actions builds and publishes to Pages.
